GHOST System Setup Instructions

Getting Started with GHOST

Follow the steps below to set up the GHOST system for use:

Prerequisites

  • Unity 2021 or later
  • ROS Noetic (for Ubuntu 20.04)
  • Required dependencies installed (refer to the repositories)

Step 1: Clone the Repositories

Clone the following repositories to your local machine:

  • git clone https://github.com/yourusername/ghost-unity.git
  • git clone https://github.com/yourusername/robotdev.git

Step 2: Set Up the Unity Environment

  1. Open the GHOST Unity repository in Unity.
  2. Import any necessary packages as specified in the documentation.
  3. Configure the scene settings according to your project needs.

Step 3: Configure ROS

Follow these instructions to set up the ROS environment:

  1. Source your ROS workspace:
    2. source /opt/ros/noetic/setup.bash
  2. Build the robotdev package:
    3. cd robotdev && catkin_make

Step 4: Running the System

To start the GHOST system, run the following commands:

  1. Launch the ROS core:
    2. roscore
  2. In a new terminal, run the robot control node:
    3. rosrun robotdev control_node
  3. Open Unity and start the scene to begin interaction.
